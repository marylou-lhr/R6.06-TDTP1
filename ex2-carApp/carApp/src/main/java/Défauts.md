# Défauts rencontrés dans l'application
## Classe Voiture
* Non présence d'un destructeur <br>
* Non présence d'un getter et d'un setter pour modele et couleur <br>
* Absence d'un setter pour vitesse <br>
## Classe Conducteur
* La méthode <i>demarrerVoiture</i> n'utilise pas son paramètre <br>
* La méthode <i>arreterVoiture</i> n'utilise pas son paramètre <br>
* Dans la méthode <i>changerVitesse</i>, la variable déclarée <i>vitesseActuelle</i> n'est pas utilisée <br>
## Classe Application
Aucun défaut trouvé

# Types d'erreurs trouvés par CheckStyle
* Fichier manquant
* Constructeur par défaut/public
* Absence de commentaire explicatif de fonction
* Erreur de type de paramètre
* Valeurs qui ne sont pas dans des variables
* Erreur d'identation/whitespace
* Confusion de conception de classe
* Ligne de code trop longue
